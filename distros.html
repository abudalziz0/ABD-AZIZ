🔹 تثبيت توزيعات Kali Linux المختلفة

📌 تثبيت Kali NetHunter (بدون روت)

اتبع الأوامر التالية لتثبيت NetHunter على Termux:

pkg update && pkg upgrade -y
pkg install wget -y
wget -O install-nethunter-termux https://offs.ec/2MceZWr
chmod +x install-nethunter-termux
./install-nethunter-termux

⚡ تثبيت Kali Linux (Proot-Distro)

يتيح لك PRoot تثبيت Kali Linux داخل Termux بسهولة:

pkg update && pkg upgrade -y
pkg install proot-distro -y
proot-distro install kali
proot-distro login kali

🚀 تثبيت Ubuntu داخل Termux

إذا كنت تريد تثبيت Ubuntu بدلاً من Kali Linux:

pkg update && pkg upgrade -y
pkg install proot-distro -y
proot-distro install ubuntu
proot-distro login ubuntu

🔹 تثبيت Debian داخل Termux

إذا كنت تفضل Debian، استخدم هذه الأوامر:

pkg update && pkg upgrade -y
pkg install proot-distro -y
proot-distro install debian
proot-distro login debian

📌 تشغيل سطح المكتب في Kali Linux

بعد تثبيت Kali، يمكنك تشغيل سطح المكتب باستخدام:

nethunter kex

أو تشغيله داخل VNC Viewer
