🔹 مكاتب وأوامر Termux الأساسية

📌 تثبيت الحزم الأساسية في Termux

  • pkg update && pkg upgrade -y - تحديث الحزم وترقيتها
  • pkg install git -y - تثبيت Git
  • pkg install python -y - تثبيت Python
  • pkg install nano -y - تثبيت محرر النصوص Nano
  • pkg install wget -y - تثبيت Wget لتحميل الملفات
  • pkg install curl -y - تثبيت Curl لجلب البيانات من الإنترنت

⚡ تثبيت أهم المكاتب والأدوات

  • pkg install openssh -y - تثبيت OpenSSH
  • pkg install php -y - تثبيت PHP
  • pkg install nodejs -y - تثبيت Node.js
  • pkg install nmap -y - تثبيت Nmap لفحص الشبكات
  • pkg install metasploit -y - تثبيت Metasploit Framework

🚀 أوامر إضافية مهمة

  • termux-setup-storage - إعطاء الصلاحيات للوصول إلى الملفات
  • ssh -V - التحقق من إصدار SSH
  • python3 --version - التحقق من إصدار Python
  • npm -v - التحقق من إصدار npm
  • ls - عرض الملفات والمجلدات

📂 إدارة الملفات في Termux

  • cd - الانتقال بين المجلدات
  • mkdir اسم_المجلد - إنشاء مجلد جديد
  • rm -rf اسم_المجلد - حذف مجلد بالكامل
  • mv ملف_قديم ملف_جديد - إعادة تسمية ملف
  • cp ملف1 ملف2 - نسخ ملف

🔹 جميع الأوامر متاحة للاستخدام التعليمي فقط 🔹
